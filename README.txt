<Auto Tagger v.1.0>

# 사용자 메뉴얼

1. 태깅할 대상 텍스트 파일과 리스트 엑셀 파일을 Auto Tagger 실행 파일이 있는 폴더 내로 옮깁니다.

2. Auto Tagger 실행 후 텍스트 파일과 엑셀 파일의 이름 (확장자 미포함)을 입력합니다.

3. Tagging 버튼을 누르면 태깅이 완료됩니다.

4. '텍스트파일명_tagged' 라는 이름으로 생성된 태깅 완료 텍스트 파일을 확인합니다.


# 주의 사항
1. 텍스트 파일의 경우 .txt 확장자를 지닌 파일이어야 하며, 엑셀 파일의 경우 .xlsx 또는 .xls 확장자를 지닌 파일이어야 합니다.

2. 엑셀 파일이 정해진 형식에 맞게 작성되어야만 정상적으로 프로그램이 동작합니다.
